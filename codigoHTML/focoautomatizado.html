Detalle del Proyecto

Conoce paso a paso cómo fue desarrollado uno de nuestros proyectos más destacados de este Semestre .

Proyecto: Automatizado de Foco de Hogar

Paso 1: Definición del problema

Se identificó la necesidad de optimizar el encendido y apagado de un foco desde alguna interfaz

Paso 2: Diseño del circuito

Se usó una placa ESP32 , un foco, un soquet y relés. Se diseñó el circuito usando Fritzing para planear las conexiones.

Paso 3: Programación

El ESP32 se programó en Arduino IDE para leer los valores del sensor y activar la bomba según la humedad detectada. También se incluyó una interfaz web.

Paso 4: Pruebas y ajustes

Las pruebas que se realizaronfueron que nos diera la ip correcta y que se manejara de manera correcta el foco

Paso 5: Resultados

El sistema si nos permite hacer el encendido y el apagado de nuestro foco desde nuestra pagina web
